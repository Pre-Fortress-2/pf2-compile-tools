# [PLEASE VISIT THE RELEASES FOR DOWNLOADS.](https://github.com/Pre-Fortress-2/pf2-compile-tools/releases)

This repo is for storing [extra large files in the releases](https://docs.github.com/en/repositories/working-with-files/managing-large-files/about-large-files-on-github#distributing-large-binaries) that cannot be normally stored in a repo.
- These files will likely be essential for compiling PF2.

### Why?
*[GitHub has a 100mb file limit for repo uploads.](https://docs.github.com/en/repositories/working-with-files/managing-large-files/about-large-files-on-github)*
